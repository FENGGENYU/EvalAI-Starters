In the final evaluation phase, the participant will generate a JSON file for submission. which should contain the predicted semantic label for each convex shape. The code to create such a JSON file from a list of NumPy arrays is provided here.

The generated JSON file should adhere to the following format, where each entry represents the prediction for one product:


[
  {
     "ASIN": "B075X33T21",
     "convex labels": [0, 1, 0, 2, 3],
  },
  {
     "ASIN": "B084DCJKSL",
     "convex labels": [0, 0, 0, 3, 3],
  },
]

To check the script for how to generate a sample json file. 


python ./submission/generate_random_submission.py -o {output_pkl_path} -g {evaluation_gallery_file} -q {evaluation_query_file}
