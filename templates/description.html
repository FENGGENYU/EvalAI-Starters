Introduction

Welcome to the ICCV 3DVeComm Workshop Challenge! This challenge focuses on the recently introduced Amazon-Berkeley-Object (ABO) dataset, which consists of high-quality, standardized 3D models of real products available for purchase online. These models have been skillfully created by artists and are composed of build-aware connected components that represent various shape properties such as texture, motion, function, interaction, and construction.

The primary objective of the workshop challenge is to assign fine-grained semantic labels (e.g., leg and arm, as defined by PartNet[5]) to the connected components in the ABO dataset. The 3D models with build-aware connected components are represented as a set of convex shapes, as illustrated in the figure below.

Participants in this challenge will be tasked with predicting the semantic labels for each convex shape. This exciting opportunity allows researchers, students, and data scientists to apply their expertise in computer vision and machine learning to address real-world e-commerce problems.

The challenge is part of the 3DVeComm workshop.

Dataset

The dataset is divided into training and validation sets for participants to train and validate their models. Each 3D convex shape comes with a ground truth semantic label. You can download the dataset from here. The code for loading and visualizing the data is available here.

The provided data file includes:

  • ASIN
  • Complete shape meshes
  • Connected components
  • Decomposed convex shapes
  • Semantic labels for each convex shape

Evaluation

The evaluation dataset has the same structure as the training dataset, but without the semantic labels:

  • ASIN
  • Complete shape meshes
  • Connected components
  • Decomposed convex shapes

To download the Evaluation dataset:
