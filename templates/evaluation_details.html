Submission Data Format

Participants are required to generate a JSON file for submission, which should contain the predicted semantic label for each convex shape. The code to create such a JSON file from a list of NumPy arrays is provided here.

The generated JSON file should adhere to the following format, where each entry represents the prediction for one product:


[
  {
     "ASIN": "B075X33T21",
     "convex labels": [0, 1, 0, 2, 3],
  },
  {
     "ASIN": "B084DCJKSL",
     "convex labels": [0, 0, 0, 3, 3],
  },
]
Participants need to submit one JSON file for development dataset and one JSON file for evaluation dataset.

Evaluation Metrics

The evaluation metric used for this challenge is the prediction accuracy. For each convex shape, the prediction accuracy will be determined by comparing the ground truth label with the predicted label. The overall prediction accuracy will be computed as the average accuracy across all convex shapes.
